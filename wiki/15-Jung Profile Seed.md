# Jung Profile — Archetypal Symbols

> **Aesthetic:** Archetypal symbols & psychological depth. 256-color palette, mystical/sacred imagery. Symbols that represent the collective unconscious: shadow, anima/animus, self, hero, wise old man, innocent, lover, magician.

The **jung** profile generates images representing Jungian archetypes: universal symbols and psychological characters that appear across myths, religions, and cultures. Visual metaphors use **sacred geometry, mandalas, and symbolic animals** to represent psychological and spiritual wholeness.

1. **Archetypal Recognition:** "What archetype does this concept represent?" (Hero, Shadow, Wise Old Man, Anima)

## Conceptual Framework: Jungian Psychology

### The Archetypes

The **fundamental patterns of human experience** that appear across all cultures and mythologies.

```
The Collective Unconscious
├── Persona (The Mask)
│   └─ Social identity, adaptation
├── Shadow (The Darkness)
│   └─ Repressed, denied, rejected aspects
├── Anima / Animus (The Opposite Gender)
│   └─ Feminine in men (Anima), masculine in women (Animus)
├── Hero (The Warrior)
│   └─ Courage, action, individuation
├── Wise Old Man / Old Woman (The Sage)
│   └─ Wisdom, knowledge, transformation
├── Innocent (The Optimist)
│   └─ Hope, faith, joy, naiveté
├── Lover (The Connection)
│   └─ Passion, intimacy, vulnerability
├── Magician (The Shaman)
│   └─ Power, knowledge, transformation
├── Shadow Guide (The Trickster)
│   └─ Chaos, illusion, necessary destruction
└── Self (The Whole)
    └─ Integration, wholeness, individuation
```

### The Journey of Individuation

```
Birth / Innocence
       ↓
Persona Formation (social identity)
       ↓
Encounter with Shadow (denied parts)
       ↓
Meeting Anima / Animus (inner opposite)
       ↓
Confrontation with Wise Old Man / Woman (wisdom guide)
       ↓
Integration & Wholeness (The Self)
```

### Symbolic Animals (Archetypal Associations)

| Archetype | Animal | Meaning |
|---|---|---|
| **Hero** | Lion, Eagle, Hawk | Courage, strength, vision, dominance |
| **Shadow** | Wolf, Serpent, Spider | Darkness, danger, hidden knowledge |
| **Wise Old Man** | Owl, Raven, Elephant | Wisdom, age, perspective, memory |
| **Anima** (in men) | Dove, Butterfly, Deer | Feminine beauty, grace, vulnerability |
| **Animus** (in women) | Bull, Stag, Dragon | Masculine power, assertion, fire |
| **Innocent** | Lamb, Butterfly, Songbird | Purity, hope, freedom |
| **Lover** | Peacock, Swan, Nightingale | Beauty, passion, expression |
| **Magician** | Raven, Snake, Fox | Transformation, trickery, knowledge |
| **Self** | Phoenix, Mandala, Ouroboros | Wholeness, renewal, integration |

```
        ║   ╲   ╲───╱   ╱   ║  Circles = integration

        ║    ╱─────────╲    ║             masculine/feminine

```

### The Shadow (Darkness with Gold)

```
Outer circle (persona, light) ────────╮
                                       │
Inner circle (shadow, darkness) ◄─────┤  Edge = meeting point
                                       │
Gold thread connecting both ◄──────────╯  Integration begins
```

## Generation Prompt Template

```
Generate SVG for the Jung profile.

Concepts: [user tags]
Archetypal focus: [optional: hero, shadow, wise-old-man, anima, self, etc.]
Journey phase: [optional: call, threshold, trial, transformation, return]

Constraints:
- Max 50 elements
- Palette: 256-color (mystical, balanced warm/cool)
- Use archetypal symbols (mandala, sacred geometry, animal)
- Integrate light and shadow (not pure light or pure darkness)
- Consider the individuation journey

Archetypal Guidelines:
- Hero concept: strong lines, gold, ascending, confident
- Shadow concept: curves, dark, inward, mysterious
- Wise Old Man concept: mandala, gold, centered, knowing
- Anima concept: curves, soft, flowing, vulnerable
- Self concept: mandala or circle, integrated, balanced

Tone: archetypal, mystical, symbolic, sacred
Example: If concepts are ['hero', 'shadow', 'self']:
  - Center: mandala representing Self (wholeness)
  - Upper quadrant: golden lion (Hero archetype)
  - Lower quadrant: serpent in darkness (Shadow archetype)
  - Integration: mandalic frame connecting all

Output: valid SVG
```
