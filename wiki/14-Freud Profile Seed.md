# Freud Profile — Layers of the Psyche

> **Aesthetic:** Layers of consciousness. Grayscale + sepia, concentric structures for id/ego/superego. Dream-like, introspective, symbolic of unconscious drives.

The **freud** profile generates images representing psychoanalytic concepts: the unconscious (id), the rational self (ego), and the internalized values (superego). Visual metaphors use **nested layers, spirals, and organic forms** to represent psychological depth.

---

## Competency Questions (Scope)

1. **Layering:** "How do I visually represent id (innermost), ego (middle), superego (outer)?"
2. **Defense Mechanisms:** "What visual metaphors represent repression, projection, sublimation?"
3. **Dream Logic:** "How do I depict surreal, symbolic, emotionally-driven imagery?"
4. **Psychological Concepts:** "Can I visualize 'shadow', 'resistance', 'transference'?"
5. **Emotional Tone:** "Does the generation feel introspective, mysterious, vulnerable?"

## Conceptual Framework: Psychoanalytic Layers

### The Tripartite Self

```
SUPEREGO (Outer Layer)
├─ Conscience, morality, social rules
├─ Guilt, shame, judgment
└─ Visual: hierarchical, restrictive, cold (gray)

EGO (Middle Layer)
├─ Reality principle, mediator
├─ Reason, defense, compromise
└─ Visual: balanced, transitional, mixed tones (gray-sepia)

ID (Inner Layer)
├─ Primal drives, pleasure principle
├─ Desire, aggression, sexuality
└─ Visual: chaotic, organic, warm (sepia/rust)
```

### Defense Mechanisms (Psychological Concepts to Map)

| Mechanism | Visual Metaphor | Example Image |
|---|---|---|
| **Repression** | Burial, darkness, erasure | Solid shape fading into black |
| **Projection** | Mirroring, reflection distortion | Image reflected in wavy surface |
| **Sublimation** | Transformation, elevation | Fire → spiral → light |
| **Regression** | Shrinking, returning to origin | Large form collapsing inward |
| **Denial** | Absence, void, negation | Empty space, broken lines |
| **Displacement** | Shifting, redirection | Arrow changing course mid-path |
| **Condensation** | Merging, overlay, fusion | Multiple forms occupying same space |

## Generation Prompt Template

```
Generate SVG for the Freud profile.

Concepts: [user tags]
Psychological theme: [id/ego/superego focus or mixed]
Defense mechanism focus: [optional: repression, projection, etc.]

Constraints:
- Max 60 elements
- Palette: grayscale + sepia only
- Use concentric layers to represent id (center, sepia/warm)
              → ego (middle, mixed)
              → superego (outer, gray/cold)
- Organic forms (spirals, tangles, webs) for id-driven concepts
- Geometric forms (grids, hierarchies) for superego-driven concepts
- Transitional forms (mirrors, doors, bridges) for ego-mediated concepts

Tone: introspective, psychological, dream-like, symbolic
Example: If the concepts are ['desire', 'guilt', 'repression'], generate:
  - Inner sepia spiral (desire, id)
  - Layered gray grid overlaying it (guilt, superego)
  - Blur or fade between them (repression, ego's defense)

Output: valid SVG
```

## How to Use with LLMs

### 1. Extraction (Freudian Concept)

```
Prompt to Claude:

You are a Freudian concept extractor. Given this ontology:
[JSON-LD above]

Task: Extract 3–5 psychological concepts from this prompt, 
using ONLY labels from the Freud ontology.

User input: "I feel trapped by expectations but desire freedom"

Output JSON: {
  "concepts": ["superego", "ego", "eros"],
  "id_layer": ["eros"],
  "superego_layer": ["guilt", "expectation"],
  "ego_layer": ["conflict", "compromise"],
  "dominant_drive": "eros (freedom vs superego restraint)",
  "suggested_defense": "conflict (unresolved tension)"
}
```

### 2. Generation with Psychological Focus

```
Generate SVG for the Freud profile.

Concepts: ["desire", "guilt", "conflict"]
Psychological interpretation:
  - desire = eros, id-driven (sepia, warm, flowing)
  - guilt = superego imposition (gray, cold, grid-like)
  - conflict = ego trying to reconcile (transitional, mirrored)

Visualize as concentric layers:
  - Inner sepia spiral: eros (desire, pleasure)
  - Outer gray grid: superego (guilt, judgment)
  - Middle blurred/transitional: ego defending against unconscious

Result: Dream-like image of psychological layering
```

### 3. Defense Mechanism Visualization

```
Concept: "repression"

From ontology: repression = downward motion, burial, erasure
Visualize as: large spiral or shape gradually fading downward into black,
             lines becoming fainter, form dissolving

Tone: somber, introspective, slightly anxious
```
